# P4 blocked vs. interleaved

This folder contains the P4 blocked-vs-interleaved test code. The experiment trains two matched causal-LM arms on the same four synthetic skill categories: rotate left, rotate right, reverse, and swap adjacent digit pairs. In the blocked arm, the model sees all examples from one skill before moving to the next. In the interleaved arm, the same examples are mixed across skills throughout training. The runnable scripts are in this directory:

- `generate_data.py` creates the deterministic train/test JSONL files and blocked/interleaved schedules.
- `make_random_olmo.py` creates the seeded random-initialized OLMo-family checkpoint used for the FarmShare run.
- `train_compare.py` trains and evaluates the blocked and interleaved arms from the same checkpoint.
- `run_farmshare.sbatch` is the Slurm wrapper used for the GPU run.

## Results

The completed FarmShare run is recorded in `results/farmshare-olmo-162m-job1654870/`. It used a random-initialized `OlmoForCausalLM` model with 162,164,736 trainable parameters. Both arms used the same checkpoint, examples, optimizer settings, and held-out test set.

| Condition | Correct | Accuracy |
|---|---:|---:|
| Base checkpoint | 0 / 1,024 | 0.00% |
| Blocked | 281 / 1,024 | 27.44% |
| Interleaved | 631 / 1,024 | 61.62% |

Interleaving improved held-out exact-match accuracy by **34.18 percentage points** over blocked training. Per skill, the interleaved arm was better on rotate-left, rotate-right, and reverse; the blocked arm was strongest on the final blocked skill, swap-adjacent-pairs.

## Datasets

The dataset is generated by `generate_data.py`; no external training corpus is required for this test. It contains 4,096 training examples per skill, for 16,384 training examples per arm. The held-out test set contains 256 examples per skill, for 1,024 total examples, with zero overlap in digit skeletons between train and test. The generator seed is `20260723`.

## Reproduce

```bash
cd p4/blocked-vs-interleaved
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
python generate_data.py --output-dir data
python make_random_olmo.py --output-dir models/olmo-162m-random
python train_compare.py \
  --model models/olmo-162m-random \
  --data-dir data \
  --output-dir outputs/local \
  --deterministic
```
